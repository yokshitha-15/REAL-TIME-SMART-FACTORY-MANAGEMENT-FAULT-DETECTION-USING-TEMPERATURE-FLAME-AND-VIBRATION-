# Real-Time Smart Factory Management and Fault Detection Using Temperature, Flame and Vibration

## Overview
This project is an IoT-based smart factory monitoring system developed using Arduino and ESP32. It continuously monitors factory conditions using temperature, flame, and vibration sensors to detect potential faults and improve industrial safety. The system provides real-time monitoring and alerts when abnormal conditions are detected.

## Features
- Real-time Temperature Monitoring
- Flame Detection
- Vibration Monitoring
- Fault Detection and Alerts
- IoT-based Monitoring using ESP32
- Automatic Safety Response

## Components Used
- Arduino Uno
- ESP32
- Temperature Sensor
- Flame Sensor
- Vibration Sensor
- Buzzer
- Relay Module
- Jumper Wires
- Power Supply

## Software Used
- Arduino IDE
- Embedded C
- ESP32 Board Package

## Working
The temperature, flame, and vibration sensors continuously collect data from the factory environment. The Arduino processes the sensor readings and identifies abnormal conditions. The ESP32 sends the monitored data for IoT-based monitoring. If any fault is detected, the system activates alerts to improve safety and prevent accidents.
